Diseña y desarrolla una página web estática completa para un partido político que busca difundir su trayectoria, logros de gobierno, propuestas y actividades de campaña de cara a una próxima elección.

Objetivo

Crear un sitio web moderno, profesional, responsive y visualmente atractivo que permita informar a los ciudadanos sobre la gestión realizada, fortalecer la imagen institucional del partido y presentar sus propuestas para un nuevo período de gobierno.

Requisitos generales
Utilizar HTML5, CSS3 y JavaScript puro.
Diseño totalmente responsive para móviles, tablets y computadoras.
Estilo moderno, institucional y confiable.
Optimización para velocidad de carga.
Código limpio y bien comentado.
Preparado para alojarse en cualquier servidor web estático.
Incorporar animaciones suaves y efectos visuales elegantes.
Cumplir buenas prácticas de accesibilidad.
